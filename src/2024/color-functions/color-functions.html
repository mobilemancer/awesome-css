Color Functions

🎨 The color() function allows you to specify colors in different color spaces

📝 Syntax: color(colorspace c1 c2 c3 [ / A])

🌈 Predefined colorspaces include srgb, srgb-linear, display-p3, a98-rgb, prophoto-rgb, rec2020, xyz, xyz-d50, and xyz-d65

🎚️ c1, c2, c3 are the component values in the color space

👻 A is an optional alpha-value

🖥️ The way these colors are displayed can depend on several factors, including the monitor’s color depth, settings, and color profile

The new color ranges

            .lab {

            .oklab {

                background-image: linear-gradient(to right in srgb-linear, blue, red);
