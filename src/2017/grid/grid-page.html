Code - Basic Grid

      .container {
        display: grid;
        grid-template-columns: 100px 200px 50px;
        grid-template-rows: 100px 100px 200px;
      }

Positioning items in the grid

      .item {
        grid-column: 2;
        grid-row: 3 / 4;
      }    
      .item {
        grid-column: 1 / span 3;
        grid-row: 2 / span 2;
      }

Code - Basic Grid, using lines

      .container {
        display: grid;
        grid-template-columns: [left-edge] 100px [center] 200px [right-edge];
        grid-template-rows: [top-edge] 100px [middle] 200px [bottom-edge]; 
      }

Positioning items in the grid

      .item {
        grid-column: left-edge / center;
        grid-row: top-edge / bottom-edge;
      }

Code - Basic Grid, auto, fr & repeat

      .container {
        display: grid;
        grid-template-columns: 1fr 3fr 1fr;
        grid-template-rows: auto 1fr 2fr auto;
      }

Code - Creating a Grid Using Named Areas

  .named-container {
    display: grid;
    grid-template-areas: "header header header"
                         "left main sidebar"
                         "footer footer footer";
  }

Code - Positioning Elements in Named Areas


  .header {
    grid-area: header;
  }

  .left {
    grid-area: left;
  } 

  .main {
    grid-area: main;
  }

  .sidebar {
    grid-area: sidebar;
  }

  .footer {
    grid-area: footer;
  }

Code - Markup for Named Area Grid

<section class="named-container"> <div class="footer">Footer</div> <div class="main">Middle</div> <div class="sidebar">Right side</div> <div class="left">Left side</div> <div class="header">Header</div> </section>

Implicit Areas

üìè Define matching lines with {identifier}-start & {identifier}-end

üìê Will create an area with the {identifier} as name

      .container {
        display: grid;
        grid-template-columns: 
          [full-start] 100px [main-start] auto [main-end] 1fr [full-end];
        grid-template-rows: [height-start] auto [height-end];
      }
    
      .item {
        grid-area: main;
      }

Responsive Grid

Use @media or @container querries to modify the grid to make it responsive 

@container (max-width: 800px) {
  .query-container {
      grid-template-areas: "header" "main" "footer";

      .left {
          display: none;
      }

      .sidebar {
          display: none;
      }
  }
}
