Code - Basic Grid

      .container {
        display: grid;
        grid-template-columns: 100px 200px 50px;
        grid-template-rows: 100px 100px 200px;
      }

Positioning items in the grid

      .item {
        grid-column: 1 / 2;
        grid-row: 2 / 3;
      }    
      .item {
        grid-column: 1 / span 2;
        grid-row: 2 / span 2;
      }

Code - Basic Grid, using lines

      .container {
        display: grid;
        grid-template-columns: [left-edge] 100px [center] 200px [right-edge];
        grid-template-rows: [top-edge] 100px [middle] 200px [bottom-edge]; 
      }

Positioning items in the grid

      .item {
        grid-column: left-edge / center;
        grid-row: top-edge / bottom-edge;
      }

Code - Basic Grid, auto, fr & repeat

      .container {

        grid-template-columns: 1fr 3fr 1fr;
        grid-template-rows: auto 1fr auto;
      }

Code - Creating a Grid Using Named Areas

  .named-container {
    background-color: purple;
    display: grid;
    gap: 20px;
    grid-template-areas: "header header header"
                         "left main sidebar"
                         "footer footer footer";
  }

Code - Named Areas, Items...


  .item1 {
    grid-area: header;
  }

  .item2 {
    grid-area: left;
  } 

  .item3 {
    grid-area: main;
  }

  .item4 {
    grid-area: sidebar;
  }

  .item5 {
    grid-area: footer;
  }

Implicit lines

Define matching lines with *-start & *-end

Will create an area with the identifier name

      .container {
        display: grid;
        grid-template-columns: 
          [full-start] 100px [main-start] auto [main-end] 1fr [full-end];
        grid-template-rows: [row-start] auto [row-end];
      }
    
      .item {
        grid-area: main;
      }

"Dynamic" Grid

Use @media or @container querries to modify the grid to make it responsive

  .query-container {
      grid-template-areas: "header" "main" "footer";

      .item2 {
          display: none;
      }

      .item4 {
          display: none;
      }
  }
}
