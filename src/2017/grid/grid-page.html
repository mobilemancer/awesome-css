Code - Basic Grid

  .container {
    background-color: coral;
    display: grid;
    gap: 20px;
    grid-template-columns: repeat(3, 1fr);
  }
<section class="container"> <div class="item">Hello 1</div> <div class="item">Hello 2</div> <div class="item">Hello 3</div> <div class="item">Hello 4</div> <div class="item">Hello 5</div> </section>

Code - Creating a Grid Using Named Areas

  .named-container {
    background-color: green;
    display: grid;
    gap: 20px;
    grid-template-areas: "header header header"
                         "left main sidebar"
                         "footer footer footer";
  }

  .item1 {
      background-color: violet;
      grid-area: header;
  }

  .item2 {
      grid-area: left;
  }

Named Areas - Cont...
