Unset

🛑 Used to "reset" a css property value

⤴️ Resets to inherited value, if it's an inheriting property

⬆️ Resets to initial value, for non inheriting properties

Markup - Using unset of different props

<div> <p>...</p> <p>...</p> <p class="no-border">...</p> </div>

CSS - Using unset of different props

      div {
          border: 1px solid red;
      }

      div>:first-child {
          border: inherit;
      }

      p {
          border: 2px solid green;
      }

      .no-border {
          border: unset;
      }

Result

Unset

We're in the first child of the div (a paragraph), we have a red border

We're in a nested paragraph, we have a Green border

We're in a nested paragraph, but border is unset
