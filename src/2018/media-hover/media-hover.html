Usage

Style for devices that support hover with hover: hover

Style for devices that don't support hover with hover: none

  @media(hover: hover) {
    /* styles for devices with hover */
  }

  @media(hover: none) {
    /* styles for devices without hover */
  }

Code - CSS

            .title {
                opacity: 0;

                @media(hover: none) {
                    opacity: 1;
                }
            }

            &:hover {
                .title {
                    opacity: 1;
