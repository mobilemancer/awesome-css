Grid-Gap ➡️ "Flex-gap"

🌅 In the begining there was grid-gap

🤔 Then came flexbox-gap

📲 Shorthand for row-gap & column-gap

Gap

1. gap: 20px 10px;
2. gap: 20px;

The first value is for row gap. The second value is for column gap, and if this is omitted, then it's assumed to have the same value as row gap.
