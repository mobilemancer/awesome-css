CSS Nesting

🎓 CSS nesting used to be a priviliege only for users of CSS pre-processors

🥰 Big part of why SASS and LESS became so popular

😃 Now available in all browsers, no build tools required!

Simple Nesting

.section-one {
    h2 {
        background: linear-gradient(to right, purple, hotpink);
        -webkit-background-clip: text;
        background-clip: text;
        -webkit-text-fill-color: transparent;
        width: fit-content;
    }
}
<h2>This is a h2 outside of the section!</h2> <section class="section-one"> <h2>Section One</h2> </section>

Result

This is a h2 outside of the section!

Complex Nesting

For combined selectors use ampersand "&".

The common sibling and child slectors are also available.

ul > li { } ⬇️ ul { & > li { } }
