CSS Nesting

🎓 CSS nesting used to be a priviliege only available for users of CSS pre-processors

🥰 Big part of why SASS and LESS became so popular

😃 Now available in all browsers, no build tools required!

Code: Simple Nesting

.section-one {
    h2 {
        background: linear-gradient(to right, purple, hotpink);
        -webkit-background-clip: text;
        background-clip: text;
        -webkit-text-fill-color: transparent;
        width: fit-content;
    }
}
<section class="section-one"> <h2>Section One</h2> </section> <section class="section-two"> <h2>Section Two</h2> </section>

Code: Complex Nesting

For combined selectors use ampersand "&".

The common sibling and child slectors are also available.

  ul > li { }

  ⬇️
  
  ul {
    & > li { }
  }
