Subgrid

📦 Subgrid is a feature of the CSS Grid Layout Module

📏 It allows a grid item to become a grid container and define its own grid

🔗 It inherits the grid definition from its parent grid

📐 It aligns the grid lines of the subgrid with the parent grid

Visualization

"16 boxes, stacked 4 vertical, 4 horizontal, whereof the 4 in the lower right corner are purple and glowing"
