Subgrid

üì¶ Subgrid is a feature of the CSS Grid Layout Module

üìè It allows a grid item to become a grid container and define its own grid

üîó It inherits the grid definition from its parent grid

üìê It aligns the grid lines of the subgrid with the parent grid

Visualization

"Please draw me 16 boxes, stacked 4 vertical, 4 horizontal, whereof the 4 in the lower right corner are purple and glowing"

Code - Markup

<div class="grid-container"> <div class="grid-item">1</div> <div class="grid-item">2</div> <div class="grid-item">3</div> <div class="grid-item">4</div> <div class="grid-item">5</div> <div class="subgrid-container"> <div class="grid-item">J</div> <div class="grid-item">P</div> <div class="grid-item">F</div> <div class="grid-item">C</div> </div> </div>

Code - CSS

            .grid-container {
                display: grid;
                grid-template-columns: repeat(3, 1fr);
                grid-template-rows: repeat(3, 1fr);
                gap: 1rem;
        
                .grid-item {
                    height: 9rem;
                    display: flex;
                    justify-content: center;
                    align-items: center;
                    background-color: blueviolet; }
        
                .subgrid-container {
                    display: grid;
                    grid-column: 2 / span 2;
                    grid-row: 2 / span 2;
                    grid-template-columns: subgrid;
                    grid-template-rows: subgrid;
        
                    .grid-item {
                        background-color: indigo;
                        box-shadow: 5px 5px 15px purple;  } } }
