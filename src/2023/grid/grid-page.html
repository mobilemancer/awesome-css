Creating a Basic Grid

  .container {
    background-color: coral;
    display: grid;
    gap: 20px;
    grid-template-columns: repeat(3, 1fr);
  }
<section class="container"> <div class="item">Hello 1</div> <div class="item">Hello 2</div> <div class="item">Hello 3</div> <div class="item">Hello 4</div> <div class="item">Hello 5</div> </section>

Creating a Grid Using Named Areas

  .named-container {
    background-color: green;
    display: grid;
    gap: 20px 0px;
    grid-template-areas: "header header header" "left main sidebar" "footer footer footer";

    .item1 {
        background-color: violet;
        grid-area: header;
    }

    .item2 {
        grid-area: left;
    }

    .item3 {
        grid-area: main;
    }

    .item4 {
        grid-area: sidebar;
    }

    .item5 {
        background-color: violet;
        grid-area: footer;
    }
  }
<section class="named-container"> <div class="item item5">Footer</div> <div class="item item4">Right side</div> <div class="item item3">Middle</div> <div class="item item2">Left side</div> <div class="item item1">Header</div> </section>

"Dynamic" Grid

Possible to use @media or @container querries to modify the grid to make it responsive 

@container (max-width: 600px) {
  .query-container {
      grid-template-areas: "header" "main" "footer";

      .item2 {
          display: none;
      }

      .item4 {
          display: none;
      }
  }
}
