Usage

Define a parent container with a Contianment Context.

.parent-container  {
  container-type: size | inline-size | normal;
}

Code - Setting Up the Container

  .card-container {
    container-type: inline-size;
  }
  
  .card {
    width: 30%;
    ...
  }

  .big {
      width: 60%;
      ...
  }

Code - Markup

<div class="card-container"> <section class="card"> <h2>Card</h2> <p> Lorem ipsum dolor sit amet consectetur adipisicing elit. </p> </section> <section class="card big"> <h2>Card Big</h2> <p> Lorem ipsum dolor sit amet consectetur adipisicing elit. </p> <p class="info">Info: Quisquam voluptatum, quibusdam, quia... </p> </section> ... </div>

Code - Container Query for the Container

      .card-container {

        @container (max-width: 1200px) {
          .big {
            width: 30%;
          }
        }
        
      }

Code - Container Query for Big Component

      .big {
        width: 60%;

        container-type: inline-size;
    
        @container (max-width: 400px) {
            .info {
                display: none;
            }
        }
    }

Cards will be modified at container width < 1200, Current container width: ${containerWidth}
