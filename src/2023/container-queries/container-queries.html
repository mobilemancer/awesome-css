Container Queries

We have used media queries for making our pages responsive.

But we had to rely on JavaScript to respond to specific element sizes.

Cards will be modified at container width = 600, Current container width: ${containerWidth} 

 
@container (max-width: 600px) {
  .big {
      width: 30%;
  }

  .card h2 {
      font-size: 1em;
  }
}
