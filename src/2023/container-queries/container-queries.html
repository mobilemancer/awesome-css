Usage

Define a parent container with a Contianment Context.

.parent-container  {
  container-type: size | inline-size | normal;
}

Code - Cards

  .card-container {
    container-type: inline-size;
  }
  
  .card {
    h2 {
        color: white;
    }

    width: 30%;
    color: white;
    background-color: hotpink;
    border-radius: 10px;
    padding: 20px;
    box-sizing: border-box;
  }

  .big {
      width: 60%;
      background-color: purple;
  }

Code - Markup

<div class="card-container"> <section class="card"> <h2>Card</h2> <p> Lorem ipsum dolor sit amet consectetur adipisicing elit. </p> <p class="info"> Quisquam voluptatum, quibusdam, quia... </p> </section> <section class="card big"> <h2>Card Big</h2> <p> Lorem ipsum dolor sit amet consectetur adipisicing elit. </p> <p class="info"> Quisquam voluptatum, quibusdam, quia... </p> </section> ... ... ... </div>

Code - Container query

@container (max-width: 800px) {
  .big {
      width: 30%;
  }

  .card {
      h2 {
          font-size: 1em;
      }

      .info {
          display: none;
      }
  }
}

Cards will be modified at container width < 800, Current container width: ${containerWidth}
