Scroll based Animations

🕰️ The Scroll-driven Animations Specification define two types of timelines

📜 Scroll Progress Timeline: Links timeline to scroll position

👀 View Progress Timeline: Links timeline to element’s position

🎬 The animation it drives is defined with the @keyframes keyword

Scroll Progress Timeline

🔄 Scroll-based timeline in CSS links the timeline to the scroll position

📏 As you scroll, the linked CSS animation advances or rewinds accordingly

Syntax

animation-timeline: scroll(root | nearest | self block | inline | x | y);

Code - Setup animation with @keyframes

@keyframes block-grow {
    from {
        background-color: purple;
        transform: scaleX(0);
    }

    to {
        background-color: hotpink;
        transform: scaleX(1);
    }
}

Code - Setup animation timeline

.content-port {
    width: 300px;
    height: 200px;
    overflow-y: scroll;

    .animated-block {
        height: 400px;

        animation-name: block-grow;
        animation-timeline: scroll(nearest block);
        animation-timing-function: linear;
    }
}

Code - Markup

<div class="content-port"> <div class="animated-block"></div> </div>

Result

View Progress Timeline

📚 scroll-timeline-name is used to define a named view progress timeline

🔄 The timeline progresses is based on the change in visibility of that ☝️ element inside a scroller

🎯 The element that the view-timeline is set on is reffered to as the Subject

📈 The visibility of the subject inside the viewport is tracked — by default, the timeline is at 0% at one edge of the viewport and 100% when at the opposite edge

🎥 The name is referenced in an animation-timeline declaration and used as the timeline progresses

🚫 If the viewport does not overflow or if the overflow is hidden or clipped, no timeline will be created

Code - Animation Setup

@keyframes bar-grow {
    from {
        background-color: black;
        opacity: 0;
        width: 0px;
    }

    to {
        background-color: rgb(79, 16, 120);
        opacity: 1;
        width: calc(100% - var(--menu-width));
    }
}

Code - Defining the subject

.my-selected-element {
    scroll-timeline-name: --page;
}

Code - Defining the scroller

.progress-hover {
    height: 3px;
    position: fixed;
    left: var(--menu-width);
    animation-name: bar-grow;
    animation-timeline: --page;
    animation-timing-function: linear;
}
