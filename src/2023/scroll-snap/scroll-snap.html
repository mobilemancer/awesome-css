Scroll Snap

📌 Allows snapping items in "place" when scrolling

🧭 Can set snap direction

🎯 Can set snap attach position

⚖️ Possible to set snap "strictnes"

⌛ Used to be done with JavaScript in the good old days

Modes: Proximity or Mandatory

📏 The `mandatory` mode in CSS scroll-snap ensures that the browser always snaps to a defined point

🎯 The `proximity` mode is less strict, allowing the content to snap to a point if it's close, but it's not required

⚠️ Note: `mandatory` can be problematic if content is larger than expected

Direction

🧭 The scroll-snap-type property defines the direction of scroll snapping

➡️ It can be x (horizontal), y (vertical), or both

Attach

🎯 The scroll-snap-align property defines the position where the snap target is located when the scroll operation finishes

📌 It can be start, center, end, or none

Code

.card-container {
  scroll-snap-type: y mandatory;
}

.card {
  scroll-snap-align: start | center | end | none;
}
