:has() Selector

👪 In effect - a parent selector

⌛ We used to have this in SCSS

🙏 Highly requested feature for a long time

Code - Using :has to select elements

CSS to select H2 with following div

  h2:has(+ div) {
    color: purple;
    margin-bottom: 0.5rem;
  }
