CSS Math

🌅 We got calc() early (2013), but needed more

↘️ The min() function helps us calculate the smallest value

↗️ The max() function helps us calculate the largest value

➡️⬅️ The clamp() function helps us make sure a value stays in bounds

Clamp

🔡 Syntax: clamp(min, preffered, max)

🌟 The preferred value will be used if it is within the min and max values

⬅️➡️ If the preferred value is outside the min and max values, the min or max value will be used

🧮 The values can be math expressions, calc() not needed

📐 The values can be any lenght expression, including nested min() and max() expressions

Examples

          .container {
              width: clamp(90%, 80vw, 1024px);
              font-size: clamp(1rem, 2.5vw, min(2rem, 32px));
          }
