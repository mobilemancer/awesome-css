↘️ The min() function helps us calculate the smallest value

↗️ The max() function helps us calculate the largest value

➡️⬅️ The clamp() function helps us make sure a value stays in bounds

Clamp

📏 The clamp() function takes three arguments

📏 The first argument is the minimum value

📏 The second argument is the preferred value

📏 The third argument is the maximum value

📏 The preferred value will be used if it is within the min and max values

📏 If the preferred value is outside the min and max values, the min or max value will be used
